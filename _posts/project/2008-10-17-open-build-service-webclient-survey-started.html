Today we started an Open Build Service Webclient online survey. We want to get more informations about the Open Build Service Webclient users, the used hardware and software and (potential) use cases.

If you use, used or want to use the OBS, please participate on the survey and help us to make a solid new version of the web client .

The survey is available on surveymonkey.com via this link.
