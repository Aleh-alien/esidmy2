Community Support

OBS is developed as open source project and uses therefore the usual Free and Open Source Software community resources for contacting the developers.

Support for administrations and installation problems are handled via the opensuse-buildservice mailing list (archive).

You can report bugs for the build service through Bugzilla.

The IRC channel #openSUSE-buildservice on freenode is also used for community support.

Professional Support and Consulting Offerings

We ourselves currently have no product or service offering around OBS. But we are cooperating with preferred Partners to allow customers using OBS in productive build environments. So if you need Consulting, Training or Support with respective Service Level Agreements please contact a Partner listed below.

from B1 Systems
